# Timer CMD

A dependency free cli based countdown timer

## Install

```
npm install -g timer-cmd
```

or run the command with npx

```
npx timer-cmd 5m30s
```

## Usage

```
Please specify time

Format:
	timer-cmd XXhXXmXXs
	timer-cmd XXmXXs
	timer-cmd XXs
Examples:
	timer-cmd 1h30m20s
	timer-cmd 3m20s
	timer-cmd 5m
```
